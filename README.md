# NSolucoes

## Versões:
- NPM 9.5.0
- NODE 18.15.0
- REACT 18.2.0
- Banco de Dados: mysql

## Como Rodar:
Importe o banco de dados no arquivo, ou, rode os comandos 
Vá até diretorio backend e rode:
~~~
    npm install
    npm start
~~~
Vá até diretorio server (frontend) e rode:
~~~
    npm install
    npm start
~~~
